### Installation ###

* `git clone https://github.com/anatoliWeb/laravel5-angular1-material-starter.gitgit remote add origin https://github.com/anatoliWeb/laravel5-angular1-material-starter.git projectname`
* `cd projectname`
* `composer install`
* `php artisan key:generate`
* Create a database and inform *.env*
* `php artisan migrate --seed` to create and populate tables
* Inform *config/mail.php* for email sends
* `php artisan vendor:publish` to publish filemanager
* 'npm install'
* Build ja and css
* 'gulp'
* 'gulp tdd'
* `php artisan serve` to start the app on http://localhost:8000/

Documentation for the framework can be found on the [Laravel website](http://laravel.com/docs) ,  [Angular website](https://angular.io/docs) .
